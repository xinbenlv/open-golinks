# Design Gallery 2026 Trends

## Overview

这是一个展示 **2026 年网页设计趋势** 的设计画廊，基于 Webflow 对当下设计方向的分析。页面展示 8 大设计哲学，每种都代表了不同的设计方向和品牌价值观。

## 8 大设计趋势

### 1. **Proprietary Effects & Styles** (专有效果与风格)
- 品牌开发独特的视觉系统和自定义动画
- 远离通用的算法设计
- **特点**: 自定义动画、品牌特有的运动语言

### 2. **Art + Advanced UI** (艺术 + 高级 UI)
- 将产品呈现为艺术作品，而非普通界面
- 古典美学与精致交互的结合
- **特点**: 美术馆式呈现、手工设计感

### 3. **Minimalist Copy** (极简文案)
- 极少化文字，充分利用白空间
- 用沉默和简洁表达自信
- **特点**: 极端简洁、高对比、清晰表达

### 4. **TL;DR Experiences** (速览体验)
- 结构化概览，用户快速理解
- 可选的深度探索
- **特点**: 商业报告式布局、多级信息架构

### 5. **Color System Expansion** (色彩系统扩展)
- 完整的色彩系统，而非单一主色
- 整个设计中部署全调色板
- **特点**: 多色系统、视觉自信、色彩一致性

### 6. **Dynamic Text Treatments** (动态文本处理)
- 动画化、大胆的排版
- 强调重要信息，使阅读有意图且有收获
- **特点**: 动画排版、强调效果、文本运动

### 7. **Guided Scrolling** (引导式滚动)
- 进度指示器和视觉路标
- 帮助用户了解位置和前方内容
- **特点**: 进度条、位置指示、前向指引

### 8. **Intentional Interactions** (有意的交互)
- 优先考虑有意义的交互而非算法效率
- 手工设计的交互体验
- **特点**: 有目的的反馈、精心设计的微交互

## 设计特点

### 视觉设计
- **主色调**: 深色背景（黑色/灰色）搭配多彩渐变
- **排版**: 现代黑体 + 精选字体组合
- **动画**: 优雅的过渡和交互动画
- **布局**: 2 列网格，响应式设计

### 交互
- Hover 效果：卡片发光和缩放
- 渐变色彩系统
- 平滑的过渡效果
- 浮动和脉冲动画

### 响应式
- 移动端: 1 列
- 平板: 2 列
- 桌面: 2 列

## 如何使用

### 访问页面
```
/design-gallery-2026-trend
```

### 选择合适的趋势
根据你的品牌和产品类型选择：

**SaaS & 工具类:**
- Proprietary Effects (品牌差异化)
- Minimalist Copy (用户专注)
- TL;DR Experiences (清晰度)
- Guided Scrolling (用户指引)

**创意 & 品牌类:**
- Art + Advanced UI (展示)
- Dynamic Text Treatments (强调)
- Color System Expansion (身份识别)
- Intentional Interactions (手工感)

## 实现细节

### 核心组件
- `TrendCard`: 展示单个趋势的卡片组件
  - 接收趋势编号、标题、描述、预览和强调色
  - 包含 hover 效果和发光动画

- 8 个 Preview 组件: 每个都演示特定的设计趋势

### 动画和效果
- **Blob Animation**: 背景的流动球体
- **Shimmer Effect**: 闪闪发光的文本效果
- **Pulse Glow**: 脉冲发光效果
- **Float Animation**: 浮动效果
- **Slide In**: 滑入动画

### CSS 变量使用
- 渐变颜色系统
- 统一的过渡时间
- 响应式断点
